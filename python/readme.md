### Python API for fmm

FMM python API is designed with Swig.

For installation of the API, please refer to (fmm-wiki)[https://fmm-wiki.github.io/docs/installation/].


  # Under the project folder
```
export CC=gcc-13
export CXX=g++-13

  mkdir build
  cd build
  cmake ..
  make -j4
  sudo make install
```
